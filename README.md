## 📌 **Project Overview**
**ERNEST BOT V1** is a powerful and versatile WhatsApp bot built with **JavaScript** and **Node.js**, utilizing the `@whiskeysockets/baileys` library. The bot enhances user interactions by providing various **AI-powered, automation, and moderation** features.
